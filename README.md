# DWH API
## Installation
### Ruby, Rails and dependencies

Install [rvm](https://rvm.io/).

Install ruby:

```rvm install ruby-2.6.3```

Instal dependencies:

```bundle install```

### Database

Application is using Postgres database.

Create database:

```rake db:create```

Run all migrations:

```rake db:migrate```

## Usage

Start server:

```rails s```
